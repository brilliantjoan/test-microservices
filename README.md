# Requirements
- Java 11
- Node 16
- Postgres
- Docker

## Starting the services
- go to root folder and run `docker-compose up`
- go to imdb-client and run npm run start
- open via website localhost:3000

### List of Services Port
Movie: 8100
Auth: 8000
Api Gateway: 8765
Naming Server: 8761
Frontend: 3000
